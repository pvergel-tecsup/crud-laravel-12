## Sobre el proyecto

El proyecto presenta dos CRUD para dos tablas: Categoria y Producto.

Estructura de las tablas
Categoria
- id
- descripcion

Producto
- id_producto
- nombre
- marca
- precio
- stock
- id_categoria
